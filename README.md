# shiftere

Chef cookbook that installs Shifter - Linux Containers for HPC

## Requirements

Requires the following cookbooks:

 - cookbook, '~> version'


### Platforms

The following platforms are supported:

- Ubuntu (>= 16.04)
- Debian (>= 9.0)
- CentOS (>= 6)

### Chef

- Chef 14.0+
